## How to use this file
When hitting a problem during /execute:
1. Write a one-sentence description of the problem
2. Record each attempt and its result
3. After 3 strikes: stop, record in this file, run /pause

When the issue is resolved:
1. Write the root cause
2. Write the fix
3. Move to Resolved section
